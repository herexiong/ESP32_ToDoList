## 环境要求
+ ESP-IDF 4.4.6  
+ ESP32S3 N16R8
+ SD卡
+ ST7796 显示器,分辨率为320*480
+ GT911电容触摸
+ LVGL V8.3
+ MAX98357模块(音频部分需要)
