# 桌面便签
桌面备忘录是一个电子显示器的备忘录，目的是完全替代纸质备忘录的作用，可在局域网下使用网页添加或删除备忘录，同时作为一个桌面摆件，兼具了天气时间等显示功能并具有DLNA局域网音乐投放的功能

## 文件介绍
### ESP32Project
用于存放ESP32的代码
### SquareLineProject
SquareLineProject是LVGL的图形化UI生成工具，此文件夹用于存放SquareLine工程文件及其导出的文件
### ToDoList
用于前期验证备忘录的存储及其存储结构
